# 0-n-1中缺失的数字


### 题目描述
一个长度为n-1的递增排序数组中的所有数字都是唯一的，并且每个数字都在范围0～n-1之内。
在范围0～n-1内的n个数字中有且只有一个数字不在该数组中，请找出这个数字。

输入: [0,1,3]
输出: 2

### 解题思路
可以使用二分查找或其他查找方法，在此使用的是二分查找   

### 代码实现
```
class Solution {
public:

    int missingNumber(vector<int>& nums) {
        int mid;
        int n=nums.size();
        int l=0;
        int h=n-1;
        
        while(l<=h)
        {
            mid=(l+h)/2;
            if(nums[mid]==mid)//判断往左还是往右的条件
                l=mid+1;
            else
                h=mid-1;
            
        }
        return l;
    }
};
```
